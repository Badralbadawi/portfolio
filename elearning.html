نظام التدريب والتعليم المستمر

منصة تعليمية مرنة وتفاعلية، طُوّرت باستخدام Laravel، تتيح تقديم محتوى متنوع يشمل فيديوهات، دورات مباشرة، مقالات، واختبارات، وتوفر أدوات قوية للتتبع والتقييم.

التقنيات المستخدمة

  • Laravel 9 + PHP 8.1
  • قاعدة بيانات MySQL / PostgreSQL
  • واجهة تفاعلية باستخدام Blade + Bootstrap
  • دعم التكامل مع Zoom / BigBlueButton للفصول المباشرة
  • نظام تقييم متكامل مع تقارير وتحليلات

الميزات الأساسية

  • إدارة غير محدودة للدورات التعليمية والفصول المباشرة
  • دعم رفع فيديوهات + تحميل ملفات تعليمية متعددة
  • نظام اختبارات متقدم مع الأسئلة التفاعلية
  • نظام تتبع التقدم ونقاط ومهام لكل متعلم
  • بيئة تعليمية متعددة المستخدمين (مدربين، متعلمين، مشرفين)

للمُدرّبين

  • رفع محتوى مرئي، مقاطع فيديو، ملفات PDF / PPT / DOC
  • إنشاء اختبارات تفاعلية ومهام
  • إدارة الحضور والتفاعل مع المتدربين

للمتعلمين

  • الوصول السهل للدروس المباشرة والمسجلة
  • المشاركة في المشاريع التفاعلية
  • الاطلاع على التقارير والتقدم الدراسي
