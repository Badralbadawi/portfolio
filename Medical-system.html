نظام التجهيزات الطبية والصيانة

نظام متكامل تم تطويره لخدمة إدارات الصيانة والمعايرة والتجهيزات والمناقصات والمخازن في المرافق الصحية، يهدف إلى أتمتة وتتبع كل العمليات الحيوية لضمان جاهزية الأجهزة الطبية بكفاءة وشفافية عالية.

التقنيات المستخدمة

  • Laravel 10 + PHP 8.2
  • MySQL / PostgreSQL
  • واجهة إدارة مبنية بـ Blade + Bootstrap
  • QR Code لتتبع المعدات
  • نظام صلاحيات متقدم

اقسام النظام

  • إدارة الصيانة
  • إدارة المعايرة
  • إدارة المناقصات والتجهيزات
  • إدارة المخازن
  • إدارة طلبات الخدمة

الميزات الرئيسية

  • جدولة آلية للصيانة الدورية والإصلاحات
  • تسجيل وتتبع عمليات المعايرة
  • أرشفة رقمية لعقود المعدات وبياناتها
  • نظام مناقصات ومشتريات شفاف
  • سجل صيانة شامل لكل جهاز
