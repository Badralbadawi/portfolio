نظام إدارة الصيدليات والمتاجر الطبية

منصة SaaS سحابية متكاملة لإدارة كافة العمليات التشغيلية للصيدليات والمتاجر الطبية، تشمل المبيعات، المخزون، العملاء، الاشتراكات، تقارير الأرباح، مع دعم متعدد اللغات وطابعات Bluetooth.

التقنيات المستخدمة

  • Laravel + Vue.js
  • MySQL + Firebase
  • Bootstrap واجهة إدارة
  • RESTful APIs + JWT
  • سيرفرات SaaS سحابية قابلة للتوسع

أقسام النظام

  • لوحة تحكم ذكية وتحليلات أداء
  • إدارة الفروع والموظفين والصلاحيات
  • نظام نقاط البيع POS
  • إدارة العملاء وسجل المشتريات
  • كتالوج المنتجات والموردين والمخزون
  • تقارير مالية وتنبؤات مبيعات
  • إشعارات وتنبيهات آنية
  • إدارة الاشتراكات ونماذج الدفع

الميزات الرئيسية

  • نظام سحابي مرن متعدد المستخدمين
  • تكامل مع بوابات الدفع الإلكترونية
  • إشعارات فورية عبر البريد وWhatsApp
  • دعم طابعات POS Bluetooth
  • واجهة استخدام متعددة اللغات والعملات
  • تحليلات أرباح وتوقعات المخزون
