منصة بث فيديو احترافية

مشروع متكامل لبث الفيديوهات (أفلام، مسلسلات، رياضة، بث مباشر) تم تطويره باستخدام Laravel. يدعم استيراد المحتوى من TMDb/IMDb، ويوفر لوحة تحكم قوية لإدارة كل جوانب المنصة بما فيها الاشتراكات، الدفع، المحتوى، المستخدمين، وغيرها.

💻 التقنيات المستخدمة

  • Laravel 10 + PHP 8.2
  • قواعد بيانات: MySQL / PostgreSQL
  • نظام بث HLS مع حماية Token-based streaming وDRM
  • الواجهة الأمامية: Blade + Bootstrap 5
  • ElasticSearch للبحث الذكي
  • تكامل مع TMDb / OMDb API لجلب بيانات الأفلام
  • بوابات الدفع: Stripe, PayPal, Razorpay, Paytm
  • بنية RESTful API وقابلة للتوسعة
  • أمان عالي مع دعم HTTPS وتكامل مع متطلبات GDPR

✨ أهم الميزات

  • 🎨 تصميم متجاوب (Responsive) مع 6 سكيمات لونية
  • ⚙️ لوحة تحكم للإدارة المتقدمة (أفلام، عروض، رياضة، بث مباشر)
  • 🗂️ استيراد تلقائي من TMDb و OMDb (IMDb)
  • 💰 دعم بوابات الدفع العالمية والمحلية (PayPal, Stripe, Razorpay, Paytm...)
  • 🧾 إدارة الاشتراكات، القسائم، الحد من الأجهزة، صلاحيات المشرفين الفرعيين
  • 📺 دعم صيغ فيديو متعددة: MP4، M3U8، DASH، YouTube، Vimeo، مع دعم Chromecast
  • 🧠 بحث ذكي باستخدام ElasticSearch + تصنيفات حسب اللغة والنوع
  • 🔐 حماية ضد SQL Injection + دعم HTTPS + تكامل GDPR
  • 🌐 دعم الصفحات الثابتة (نبذة عنا، سياسة الخصوصية...)
  • 🧩 إعدادات الموقع (اسم، شعار، توقيت، العملة، البريد...)
