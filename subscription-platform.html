منصة اشتراكات المحتوى

نظام SaaS متكامل لبث الفيديو وإدارة الاشتراكات، تم تطويره لتوفير تجربة مشاهدة سلسة وآمنة للمستخدمين، ويضم مجموعة من الأدوات المتقدمة لإدارة المحتوى، الحماية، والدفع الإلكتروني.

التقنيات المستخدمة

  • Laravel 9 + PHP 8.1
  • Vue.js + Bootstrap + Sass
  • قاعدة بيانات MySQL
  • نظام بث HLS وM3U8 مدعوم بالحماية
  • RESTful API للتكامل مع تطبيقات أخرى
  • بوابات دفع متنوعة: Stripe, PayPal, Razorpay
  • نظام مصادقة مزدوجة (2FA) ومنع مشاركة الحسابات
  • دعم الترجمة والتعدد اللغوي (i18n)

الميزات الأساسية

  • دعم PWA وتطبيق متجاوب للأجهزة المختلفة
  • مشغل فيديو احترافي مع خيارات جودة متعددة
  • نظام حماية من مشاركة الحسابات وتقييد الأجهزة
  • دعم البث المباشر وتشغيل الفيديوهات من روابط HLS و m3u8
  • خطط اشتراك مرنة + دعم بوابات الدفع الإلكترونية (Stripe, PayPal...)

إدارة المحتوى

  • استيراد المحتوى تلقائيًا من TMDB
  • إدارة مكتبة فيديوهات داخلية وتخصيص المحتوى
  • دعم رفع وتحويل الفيديوهات تلقائيًا

لوحة التحكم والتحليلات

  • لوحة تحكم للمستخدم والمشرف
  • إعدادات الحساب وتخصيص واجهة الاستخدام
  • تقارير مشاهدة وتحليلات مالية مدمجة

توافق وتوسّع

  • يدعم أكثر من 21 لغة (RTL وLTR)
  • متوافق مع SEO وGoogle Analytics وAdSense
